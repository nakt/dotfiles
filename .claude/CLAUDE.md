## 構造

## 利用可能な Rules

| Rule             | 対象           | 説明                           |
| ---------------- | -------------- | ------------------------------ |
| git-workflow | All files         | コミット規約、ブランチ戦略、PR ガイドライン |
| markdown     | `/*.md`      | Markdown 作成時のスタイルガイド |
